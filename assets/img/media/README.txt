THƯ MỤC ẢNH/VIDEO TẢI LÊN TỪ CMS
=================================

Khi bạn tải ảnh/video lên trong trang quản trị (admin.html → tab "Thư viện ảnh"),
ảnh được lưu tạm trong trình duyệt để xem trước.

ĐỂ ĐĂNG LÊN WEBSITE THẬT:
1. Vào tab "Thư viện ảnh" trong admin.
2. Bấm "Tải" ở từng ảnh/video → trình duyệt tải file về (thư mục Downloads).
3. Chép các file đó vào CHÍNH thư mục này: assets/img/media/
4. Quay lại admin, bấm "Xuất file data.js" → chép đè vào assets/js/data.js
5. Đăng lại website (hoặc kéo–thả lại thư mục lên Netlify).

Tên file phải GIỮ NGUYÊN như khi tải về (ví dụ: anh-du-an-1.jpg).
